# Ashii-s-Aracde
A c# console application for my retro games sql database

This was a fun little side project I did after my chosen school subject of Database Programming.

After having already made a retro games database in MS SQL. I had already added all Nintendo and Atari games/consoles I had directly in the SQL database, I had made the .txt's with the rest of the games and consoles for "ease of insert" via the console app.
